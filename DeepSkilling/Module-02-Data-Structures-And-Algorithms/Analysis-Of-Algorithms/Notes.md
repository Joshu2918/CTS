# Analysis of Algorithms

## Objective
Analysis of Algorithms is the process of evaluating the efficiency of an algorithm in terms of time and space.

## Why Analyze Algorithms?

- Compare different algorithms.
- Select the most efficient approach.
- Reduce execution time.
- Optimize memory usage.

## Time Complexity

Time complexity represents the amount of time required by an algorithm as the input size increases.

### Common Time Complexities

- O(1) - Constant Time
- O(log n) - Logarithmic Time
- O(n) - Linear Time
- O(n log n) - Linearithmic Time
- O(n²) - Quadratic Time
- O(2ⁿ) - Exponential Time
- O(n!) - Factorial Time

## Space Complexity

Space complexity is the amount of memory used by an algorithm.

## Cases

Best Case
- Minimum running time.

Average Case
- Expected running time.

Worst Case
- Maximum running time.

## Conclusion

Algorithm analysis helps in choosing efficient algorithms that improve application performance.
