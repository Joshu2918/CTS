# Exercise 5 - Return Data From Stored Procedure

## Objective

Create a stored procedure that returns employee details based on department.

## Problem Statement

Develop a stored procedure that accepts a department name as input and returns all employees working in that department.

## Concepts Used

- CREATE PROCEDURE
- Parameters
- SELECT
- WHERE
- EXEC

## Learning Outcome

Learn how stored procedures can be used to retrieve filtered data efficiently.
