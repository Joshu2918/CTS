# Exercise 2 — Implementing the Factory Method Pattern

## Objective

Implement the Factory Method pattern to create objects without exposing object-creation logic to the client.

## Concepts Used

- Factory Method pattern
- Abstraction
- Polymorphism
- Loose coupling

## How to Run

```text
java FactoryMethodDemo.java
```

## Learning Outcome

After completing this exercise, I understood how to delegate object creation to a factory class and reduce dependency on concrete implementations.
