# SOLID Principles

SOLID is a set of five object-oriented design principles for maintainable, flexible, and testable software.

## Principles

1. [Single Responsibility Principle](./SRP)
2. [Open/Closed Principle](./OCP)
3. [Liskov Substitution Principle](./LSP)
4. [Interface Segregation Principle](./ISP)
5. [Dependency Inversion Principle](./DIP)

Each section contains a runnable Java example and detailed notes.
