# HTML5 Notes

## Basic Structure

An HTML5 document starts with `<!DOCTYPE html>` and contains the `html`, `head`, and `body` elements.

## Common Elements

- Headings: `h1` through `h6`
- Paragraphs: `p`
- Links: `a`
- Images: `img`
- Lists: `ul`, `ol`, and `li`
- Tables: `table`, `tr`, `th`, and `td`
- Forms: `form`, `label`, `input`, and `button`

## Semantic Elements

`header`, `nav`, `main`, `section`, `article`, and `footer` describe the purpose of page content.

## HTML5 APIs

- Web Storage saves small amounts of data in the browser.
- Geolocation requests the user's approximate location with permission.
- Audio and video elements provide native media playback.

## Good Practices

- Add meaningful page titles and image alternative text.
- Associate every form field with a label.
- Use semantic elements to improve accessibility.
- Include the viewport meta tag for mobile devices.
