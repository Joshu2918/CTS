# C# Notes

## Features

- Object-oriented
- Type-safe
- Platform-independent with .NET

## OOP Concepts

- Encapsulation
- Inheritance
- Polymorphism
- Abstraction

## Collections

Common generic collections include:

- `List<T>` for ordered items
- `Dictionary<TKey, TValue>` for key-value pairs

## LINQ

LINQ (Language Integrated Query) provides a consistent way to query collections and other data sources.

## Async Programming

The `async`, `await`, and `Task` keywords support non-blocking asynchronous operations.
